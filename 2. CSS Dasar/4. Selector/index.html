Perbedaan ID Dan Class

ID

  1. sebuah element HTML hanya boleh memiliki 1 id
  2. setiap halaman boleh memiliki 1 elemen dengan id tersebut
  3. dapat digunkan sebagai penanda halaman pada link
  4. digunakan juga pada javascript
  5. sebaiknya tdk digunakan untk css (lebih baik gunakan class)

CLASS

Class merupakan selector yang dapat digunakan oleh lebih dari satu tag html. Artinya beberapa tag html bisa menggunakan css dari class tersebut.
